# Dialog

Modal overlay for focused tasks.

## Examples

### Open and close

Control visibility with `useState` and wire `isOpen` / `onOpenChange` to `DialogTrigger`. There is no `@var-ui/astro` Dialog — Astro and HTML previews show a static core-recipe approximation of the open chrome; full interaction is React-only.
