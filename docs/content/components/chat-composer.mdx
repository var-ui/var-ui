# ChatComposer

Bordered composer chrome around the message input and trailing actions. Pair with `ChatComposerInput` for the auto-resizing textarea and `ChatSendButton` for send/stop.

## Examples

### Default

<Demo code={`import { ChatComposer, ChatComposerInput, ChatSendButton } from '@var-ui/react';

<ChatComposer actions={<ChatSendButton onPress={send} />}>
  <ChatComposerInput value={value} onChange={setValue} onSubmit={send} />
</ChatComposer>`}>
  <ChatComposerDemo />
</Demo>

## ChatComposerInput

| Prop          | Type                      | Default             |
| ------------- | ------------------------- | ------------------- |
| `value`       | `string`                  | (required)          |
| `onChange`    | `(value: string) => void` | (required)          |
| `onSubmit`    | `(value: string) => void` | (required)          |
| `placeholder` | `string`                  | `'Type a message…'` |
| `maxRows`     | `number`                  | `8`                 |
| `isDisabled`  | `boolean`                 | `false`             |
| `className`   | `string`                  | —                   |

Enter submits (trimmed, non-empty only); Shift+Enter inserts a newline.
