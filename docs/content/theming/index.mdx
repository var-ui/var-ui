# Theming

var-ui themes are TypeStyles recipes composed into a design system. Switch light and dark mode from the header to preview how tokens respond.

## Topics

- **Design tokens** — semantic colors, spacing, and typography scales
- **Themes** — extending and overriding the default theme

Theming guides and a live theme gallery are planned for a later milestone.
