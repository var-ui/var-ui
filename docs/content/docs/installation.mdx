# Installation

var-ui ships as workspace packages in this monorepo. Downstream apps install the published `@var-ui/*` packages from npm once releases are available.

## Workspace apps

Link `@var-ui/core`, `@var-ui/react`, and `@var-ui/icons` with your package manager's workspace protocol, then add Vite aliases if you want to develop against source.

## Peer dependencies

React 19 and a modern bundler with CSS extraction support for TypeStyles are required.

Package manager commands and version pins will be documented here before the first public release.
