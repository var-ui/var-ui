# Select

Dropdown single-select with options.

## Examples

### Default

<Demo code={`import { Select } from '@var-ui/react';

<Select
  label="Fruit"
  options={[
    { id: 'apple', label: 'Apple' },
    { id: 'plum', label: 'Plum' },
  ]}
/>`}>
  <SelectDemo />
</Demo>

### Options list

Pass an `options` array of `{ id, label }` objects. Use `placeholder` when no value is selected.

<Demo code={`import { Grid, Select } from '@var-ui/react';

const options = [
{ id: 'apple', label: 'Apple' },
{ id: 'plum', label: 'Plum' },
{ id: 'orange', label: 'Orange' },
{ id: 'grape', label: 'Grape' },
];

<Grid columns={2} gap="lg">
  <Select label="Fruit" options={options} />
  <Select label="Favorite fruit" placeholder="Choose a fruit…" options={options} />
</Grid>`}>
  <SelectOptionsDemo />
</Demo>
