# ChatMessageBubble

The chat bubble — a styled content container that reads `sender` from an ancestor `ChatMessage` to auto-style its background and text color. var-ui does zero markdown parsing; pass plain text or your own rendered tree as `children`.

## Examples

### Default

<Demo code={`import { ChatMessage, ChatMessageBubble } from '@var-ui/react';

<ChatMessage sender="assistant" name="Navi">
  <ChatMessageBubble>Hello! How can I help?</ChatMessageBubble>
</ChatMessage>`}>
  <ChatMessageBubbleDemo />
</Demo>

Use `group` when rendering several consecutive bubbles from the same sender, to tighten the corners between them.
